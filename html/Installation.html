GitHelp !

Pour l'installation vous aurez besoins de :

- Git scm (ligne de commande) et Sourcetree (GUI) :

- Votre éditeur de texte préféré.

- Votre compte GitLab.

P.S. : Tout est faisable en ligne de commande.


Installation de Sourcetree


- Passez la première étape.


- Identifiez-vous comme ci-dessous : NOM Prénom et prénom.nom@cstb.fr


- Dites non à la clé SSH.


- Connectez-vous au dépots distants en vous dirigeant dans remote puis ajouter un compte.


- Configurez les champs comme ceci :
Service d'hébergement : GitLab EE
URL de l'hôte : https://scm.cstb.fr/cape
Protocole préféré : HTTPS Puis cliquez sur Rafraîchir Personal Access Token.


- Connectez vous sur GitLab pour configurer le Personal Access Token. Rendez-vous dans settings puis Acces Token.


- Configurez votre PAT et faites attention de bien le conserver.


- Authentifiez-vous sur Sourcetree avec votre identifiant GitLab et le PAT comme mot de passe.
